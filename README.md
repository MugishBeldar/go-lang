# go-lang
## Installation of go language

- install go using gvm (go vesion manager) same as nvm (node version manger)

## Check go language is install or not
```go version```   

## Go program running methods

### Method 1: Run directly
```go run hello.go```

### Method 2: Build an executable first, then run it
```go build hello.go```

```./hello```

### Method 3: Build with custom output name
```go build -o myprogram hello.go```

```./myprogram```
